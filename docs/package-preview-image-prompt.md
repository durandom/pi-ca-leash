# Package Preview Image Prompt

⚠️ Avoid official Claude, Anthropic, OpenAI, or Codex logos/wordmarks in the final image. Better: imply two distinct coding runtimes with abstract agent forms.

## Recommended concept

Pi as the orchestrating brain, with two retained coding agents under control.

### Prompt

Create a clean, modern hero image for an open-source AI coding extension called `pi-ca-leash`.

Visual idea: a glowing Greek pi symbol (`π`) acts like a calm orchestrator in the center of a dark terminal-themed scene. From the pi symbol extend two subtle luminous control lines or leashes to two distinct abstract coding-agent figures. One agent should feel Claude-like: warm, thoughtful, amber/orange, notebook or reasoning vibe. The other should feel Codex-like: cool, fast, blue/cyan, terminal or implementation vibe. Both are clearly powerful but coordinated, not wild.

The overall message should be: "these are not just models; they are full coding harnesses, and pi coordinates them like a human brain directing multiple coding agents."

Style:
- dark cinematic terminal aesthetic
- elegant, minimal, high contrast
- sharp vector-like forms with subtle depth
- premium open-source developer tooling vibe
- no photoreal humans
- no mascots
- no comedy
- no clutter
- no visible text except maybe a subtle `π` symbol
- no corporate logos or trademarked icons

Composition:
- wide banner-friendly composition
- central `π` symbol as anchor
- left and right agent balance
- enough negative space for cropping to package gallery cards
- readable at small thumbnail size

Color palette:
- charcoal / near-black background
- soft terminal green accents very lightly in background
- warm amber/orange for one runtime
- cool cyan/blue for the other runtime
- white/silver highlights on the pi symbol

Details that can appear subtly:
- faint code windows
- terminal grid lines
- thin data-flow arcs
- session/history/orchestration feel
- sense of retained worker state, not one-shot chat bubbles

Mood:
- controlled
- intelligent
- technical
- multi-agent orchestration
- local power tool for developers

## Negative prompt

Do not include:
- official Anthropic, Claude, OpenAI, or Codex logos
- product screenshots
- text-heavy overlays
- anime style
- cartoon dog leashes
- goofy humor
- crowded UI panels
- bright white background
- generic robot army imagery
- handcuffs, muzzles, cages, or hostile restraint imagery

## Safer variant

If the leash metaphor feels too aggressive, replace the leash/control lines with:
- luminous orchestration threads
- conductor-like motion trails
- linked session arcs
- subtle tether lines between pi and the two runtimes

## Suggested output targets

- primary image: 1600x900
- safe crop area centered for narrower package cards
- export PNG
